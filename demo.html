PNDA Pre-Configured Demo

Before you start

PNDA is open source Platform for Network Data Analytics, which aggregates data like logs, metrics and network telemetry. This scales up to consume millions of messages per second and manages lifecycle of applications that process and analyze data.

First, you could read a short tutorial and then you will have four hours to work with PNDA using several test datasets.

Preloaded datasets include a few tables consisting of billions of rows with 55GB - 546GB uncompressed data.

All datasets are located on a single node demo server at KodiakData MemCloud. This server is a perfect fit for this demo, even though it’s not the fastest server KodiakData provides. Demo server has 16 vCPUs, 32GB RAM and 1TB of NVMe storage.

Getting started

In order to access the demo, you need to sign up at CloudWorks portal for PNDA.

  1. Open Cloudworks Portal for PNDA http://PNDA.memcloud.works
  2. Click “Sign up for CloudWorks” and create an account.
  3. Log in to Portal. “Welcome to PNDA!” message will appear.
  4. Enjoy working with PNDA.

Please feel free to contact us if you have any questions.
